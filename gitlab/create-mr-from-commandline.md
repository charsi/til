# Gitlab - Create Merge Request From The Commandline

Just in case you are feeling extra lazy and don't want to use the web ui, try this - 

`git push -o merge_request.create -o merge_request.target=main`

More options here - 
https://docs.gitlab.com/ee/topics/git/commit.html#push-options-for-merge-requests
